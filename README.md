# Service Console Admin

사내 서비스 운영을 위한 웹 기반 관리자 콘솔입니다.  
운영자가 사용자, 권한, 서비스 상태를 효율적으로 관리할 수 있도록 설계된 관리 시스템입니다.

---

## 프로젝트 배경

대부분의 사내 서비스는 사용자 관리, 권한 제어, 운영 현황 확인을 위한  
관리자 페이지가 필수적이지만, 실제 구현은 서비스마다 제각각입니다.

본 프로젝트는 **실무에서 자주 사용되는 관리자 콘솔의 공통 기능을 정리하고**,  
이를 **모던 웹 스택(Next.js + NestJS)** 기반으로 직접 설계·구현하는 것을 목표로 합니다.

---

## 프로젝트 목표
- 실무에서 활용 가능한 관리자 콘솔 구조 설계
- 프론트엔드와 백엔드가 명확히 분리된 아키텍처 구성
- 인증 및 권한 기반 접근 제어 구현
- 유지보수와 확장성을 고려한 코드 구조 설계

---

## 기술 스택
### Frontend
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

### Backend
- NestJS
- TypeScript
- REST API
- JWT 기반 인증
- Role 기반 접근 제어 (RBAC)

---

## 주요 기능

- 관리자 인증 및 로그인
- 사용자 관리 (조회, 등록, 수정, 비활성화)
- 역할(Role) 및 권한 관리
- 관리자 전용 대시보드
- 운영 로그 조회

---
